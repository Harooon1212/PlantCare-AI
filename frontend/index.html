AI Architecture

Model Information

Built using state-of-the-art deep learning for high-accuracy plant disease classification.

🧠

Deep Learning Model

MobileNetV2 based on transfer learning with fine-tuned classification layers.

TensorFlow Keras CNN
🖼️

Image Preprocessing

Images are resized to 224×224, normalized to [0,1] range, and augmented during training.

224×224 px Normalization Augmentation
⚙️

Training Process

Trained over 40 epochs using Adam optimizer with categorical cross-entropy loss and early stopping.

Adam Optimizer 40 Epochs Early Stopping
📊

Performance Metrics

Evaluated using accuracy, precision, recall, and F1-score on the held-out test set.

About

About This Project

A final semester project combining AI and agriculture to fight plant diseases.

🎓

Semester Project

Developed as a final year computer science project applying deep learning and computer vision to solve a real-world agricultural problem.

🌱

Problem Being Solved

Plant diseases cause up to 40% crop loss worldwide. Early automated detection can save harvests and support farmers without expert knowledge.

🔬

Technology Stack

HTML, CSS, JavaScript for the frontend. Python, Flask, TensorFlow, and Keras power the backend AI prediction engine.
